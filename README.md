# Задание №3 — класс «Стек»

## Описание задания
Необходимо реализовать класс **«Стек»**.

### Свойства стека
- **Текущий элемент** — элемент, находящийся на вершине стека.
- **Количество элементов** — число элементов, которые сейчас находятся в стеке.
- **IsEmpty** — признак того, пуст ли стек.

### Методы стека
- **Добавление элемента в вершину стека**.
- **Удаление элемента с вершины стека** — метод должен возвращать удалённый элемент и убирать его из стека.
- **Очистка стека**.

## Описание приложения
Приложение демонстрирует работу структуры данных **стек** и позволяет:
- ввести значение и добавить его в стек;
- посмотреть содержимое стека сверху вниз;
- увидеть текущее количество элементов;
- получить верхний элемент стека;
- удалить верхний элемент стека.

## Что используется в проекте
- **C#**;
- **.NET 8**;
- **Avalonia UI**

## Структура проекта
- `Models/MyStack.cs` — реализация класса стека;
- `ViewModels/MainWindowViewModel.cs` — логика взаимодействия интерфейса со стеком;
- `Views/MainWindow.axaml` — пользовательский интерфейс;
- `Program.cs` — точка входа в приложение.

## Инструкция по запуску
### 1. Требования
Перед запуском убедитесь, что установлены:
- **.NET SDK 8.0** или выше.

Проверить установленную версию можно командой:

```bash
dotnet --version
```

### 2. Восстановление зависимостей
В корневой папке проекта выполните:

```bash
dotnet restore
```

### 3. Запуск приложения
Для запуска приложения используйте команду:

```bash
dotnet run
```

### 4. Сборка проекта
Если нужно только собрать проект без запуска:

```bash
dotnet build
```

## Как пользоваться приложением
1. Введите значение в поле ввода.
2. Нажмите **Push**, чтобы добавить элемент в стек.
3. Нажмите **Peek**, чтобы посмотреть верхний элемент.
4. Нажмите **Pop**, чтобы удалить верхний элемент.
5. В нижней части окна отображается текущее содержимое стека и количество элементов.
